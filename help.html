Quick Start

  1. Enable Extension: Click the Requestly icon and toggle it ON
  2. Add Rule: Right-click icon ‚Üí Options ‚Üí Add Rule
  3. Configure: Set URL pattern and action type
  4. Test: Visit your website and check the logs

URL Patterns

  • Exact: https://api.example.com/users
  • Prefix: https://api.example.com/
  • Regex: https://api\.example\.com/users/\d+

Action Types

  • Replace: Replace entire response with custom JSON
  • Patch: Modify specific fields (e.g., user.name)
  • Delay: Add artificial latency in milliseconds
  • Status: Force specific HTTP status codes

Common Use Cases

Testing: Mock API responses during development

Feature Flags: Enable/disable features dynamically

Error Simulation: Test error handling with forced errors

Performance: Simulate slow networks with delays

Troubleshooting

  • Rule not working? Check URL pattern matches exactly
  • JSON errors? Use the Format JSON button to validate
  • Can't see logs? Refresh the page after adding rules
